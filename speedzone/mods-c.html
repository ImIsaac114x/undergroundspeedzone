Mods

En esta sección encontrarás mods lorefriendly de Need For Speed Carbon.

NFSC Extra Options

Este mod añade nuevas opciones y configuraciones avanzadas para Need For Speed Carbon, permitiendo una experiencia de juego más personalizada y completa.

NFSC Unlimiter

Este mod desbloquea limitaciones del juego, permitiendo usar más autos y funcionalidades personalizadas en Need For Speed Carbon.

NFSC HD User Interface

Este mod mejora la interfaz gráfica del juego con elementos en alta definición, dando un aspecto más moderno y limpio a los menús y HUD de Need For Speed Carbon.

NFSC Widescreen Fix

Este mod corrige la resolución y la interfaz del juego para adaptarlo a pantallas panorámicas actuales.
